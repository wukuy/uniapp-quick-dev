# uniapp项目快速开发模版
# 使用
使用模版
```bash
npx degit wukuy/uniapp-quick-dev my-project
```
安装依赖
```bash
npm i
```

开发
```bash
npm run dev:h5
```

# 技术栈
 Vite + Vue3 + uView 2.x

# 功能
+ 多环境
    + test、dev、prod
+ 请求封装
    + 上传
    + 请求
    + 拦截
+ model层
    + 将所有接口放到model层，方便维护
+ utils
+ eslint
+ 全局方法
+ 分包
+ uview
+ Page组件
    + 页面标准化，定义页面头、页面内容、页面顶部
+ Scroll(下拉刷新，上拉加载)
